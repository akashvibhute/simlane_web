Authorization Failed

There was an issue adding the bot to your server. Please try again or check the troubleshooting tips below.

Important

You must have "Manage Server" permissions in your Discord server to add the bot. If you don't have these permissions, ask a server administrator to complete this process.

Troubleshooting Tips

  • Make sure you're logged into the correct Discord account
  • Verify you have "Manage Server" permissions on the target server
  • Check if your server already has the maximum number of bots (100)
  • Ensure pop-ups are enabled in your browser
  • Try using an incognito/private browsing window
  • If the bot appears offline, it may take a few minutes to come online
